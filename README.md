# proyecto

El proyecto consiste en generar una herramienta de consulta para personas en busca de adquirir una vivienda. En esta etapa del proyecto se utiliza los datos del Censo de Población y
Vivienda 2020 del INEGI que dada su metodologia nos brinda beneficios como los niveles de desagregacion y factores de conversión por lo que obtenemos
una base de datos del stock inmobiliario.
